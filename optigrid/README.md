# OptiGrid - Energy Optimization Add-on

Energy optimization add-on for Home Assistant with battery management and deferrable load scheduling.

## Features

- Battery charge/discharge optimization
- Deferrable load scheduling (EV charging, etc.)
- Solar PV integration
- Dynamic electricity pricing support
- Historical data analysis
- REST API for automation integration

## Installation

1. Add this repository to your Home Assistant add-on store
2. Install the OptiGrid add-on
3. Configure your battery and load settings
4. Start the add-on

## Configuration

The add-on runs a REST API on port 8000. Configure optimization parameters via the API endpoint.

See the [documentation](https://github.com/ViniTheVini/ha-addon-optigrid) for API details.
